# Project Plan

## Product

Tulay is an integrated management platform for store managers who sell across multiple ecommerce channels such as Etsy and Shopify.

The platform centralizes inventory, listing creation, order fulfillment, print-on-demand workflows, analytics, and AI-assisted content generation.

## Current Goal

Build a front-end MVP using mock data. Do not implement real API integrations yet.

## Core Areas

1. Dashboard
   - Revenue overview
   - Store alerts
   - Top-selling products

2. Inventory
   - Product catalog table
   - Platform publishing status
   - Sync status
   - Performance metrics

3. Listing Studio
   - Product media upload area
   - Listing metadata form
   - Platform-specific settings
   - AI-generated title, description, and tags
   - SEO scoring
   - Publishing channel readiness

4. Marketing & SEO
   - Keyword rank tracking
   - Optimization opportunities
   - AI-assisted fixes

5. Orders & Fulfillment
   - Order management
   - POD status tracking
   - Printify/Gelato workflow actions

6. Analytics & Finance
   - Revenue, expenses, profit, margin
   - Sales by channel
   - Expense breakdown
   - Product profitability

7. Settings
   - Sales channel integrations
   - POD provider integrations
   - AI settings
   - Global listing rules

## Front-End MVP Rule

Use mock data only. External integrations must be represented as disabled or placeholder states.

## Future Integrations

- Etsy API
- Shopify API
- Printify API
- Gelato API
- Local fine-tuned LLM API
